Border-Styles

The border-style property may be specified using one, two, three, or four values.

  • When one value is specified, it applies the same style to all four sides.
  • When two values are specified, the first style applies to the top and bottom, the second to the left and right.
  • When three values are specified, the first style applies to the top, the second to the left and right, the third to the bottom.
  • When four values are specified, the styles apply to the top, right, bottom, and left in that order (clockwise).

CSS Hierarchy (Inline, Internal, External)

In order of importance
  • In line style tags on a page will override Internal CSS (in the header)
  • Internal CSS in the header of a specific page will override the global (External) CSS
  • If no other style indicators exist, the global setting will be displayed

CSS Syntax


  • Best practice is to alphabetize properties within an element (color, font, size, etc)
  • Basic Syntax and Key Words reference on MDN.
    • Key words with a semi colon in front are pseudo classes, such as :hover.
  • ID vs Class
    • ID can only have one of that named ID on a page, class is not limited.
    • Use ID where you have a single element on a page to influence. (why is this useful?)
    • An HTML element can only have one ID, but it can have multiple classes assigned.
    • ID's cannot have multiple ID's identified for the same element the same way multiple classes can be combined.
  • Pseudo classes - state driven
    • State driven things such as mouseover can change the class being used
